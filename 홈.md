# 🏠 PPT 제작 볼트

마크다운(Marp)으로 발표자료를 만드는 **전용 볼트**. (문서용 메인 볼트와 분리됨)

## 🚀 새 발표 시작
Claude Code 채팅에서:
```
/brainstorm <발표 주제>
```
→ 같이 기획(brief) → `/generate` 병렬 생성 → `/slide`·`/content`·`/design`·`/diagram`·`/animate`·`/notes` 다듬기 → `/review` 검토 → `/build` 출력

## 📁 발표 목록 (decks)
- 🎨 **example** — 레이아웃·컴포넌트·테마 카탈로그(살아있는 스타일가이드) · `decks/example/` → `node build.mjs example`(기본 tech, 다른 테마는 `example <테마>`)

> 새 발표가 생기면 여기에 `[[brief]]` 링크를 추가한다.

## 📌 자주 보는 노트

- [[CLAUDE|📋 작업 규칙·스킬]]

## 🛠 빌드
```bash
node build.mjs <발표이름>        # HTML
node build.mjs <발표이름> pdf    # PDF / pptx
```
별칭(선택): `ppt <덱>`(빌드) · `pv <덱>`(라이브 뷰어·포트 1122) · `pptime <덱>`(예상 발표시간). 덱 이름 Tab 자동완성.

## ⚙️ 볼트 설정 (1회)
설정 → 파일 및 링크 → **제외된 파일**에 추가: `node_modules`, `dist`, `.build`
